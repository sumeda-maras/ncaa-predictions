# NCAA Model Prediction
Develops a machine learning model to predict NCAA Tournament game outcomes using historical team performance and basketball analytics. The model combines regular season statistics, tournament history, and ranking data to estimate the probability of each team's victory.

## Dataset
The model is trained using the Kaggle March Machine Learning 2024 dataset, which includes:
- NCAA regular season results
- NCAA tournament results
- Tournament seedings
- Team statistics
- KenPom (Pomeroy) rankings and Massey Ordinals

## Objectives
- Predict NCAA Tournament game outcomes
- Engineer features from historical team performance
- Train and evaluate predictive machine learning models
- Generate tournament win probabilities for unseen matchups
